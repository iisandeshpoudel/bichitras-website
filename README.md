## Local Development Setup

### Prerequisites
- Node.js 18+
- npm or yarn

### Getting Started

1. Clone the repository:
```bash
git clone https://github.com/bichitras/bichitras.git
```
2. Navigate to the project directory:
```bash
cd bichitras
```
3. Install dependencies:
```bash
npm install
```
4. Start the development server:
```bash
npm run dev
```
